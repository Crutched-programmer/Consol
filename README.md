Here are the following features of this prototype:
- GPIOs 3 - 10 are used for numeric keypresses (ie: 1, 2, 3, 4, 5, 6, 7, 8) (Found at the center of the image with the parallel red jumpers ). These are used byt the WebTV page to reveal tiles.(Check the cartridges.md file from the details folder for better understanding)
- GPIO 2 is used as a common GPIO for the four IR recievers ( bottom of the picture )
- PIN A4, A5, A6 are used for three buttons ( To the left of the red jumpers, labelled X, Y, B, and connected by the orange jumpers) these are direct buttons for the first prototype to test out the webhooks I had created ( check the webhooks.md and the cartidges.md file in the details folder for better understanding and inspiration)

The prototype works well so far with no issues at all. I am expecting ver 0.1.4 to be completed by 12th of March 2026 with minor asthetic and functional changes to the hardware design and functional changes to the circuitry. Lets hope I dont mess up.😎
